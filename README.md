# MedCare

MedCare is a React-based web application. It serves as a modern front-end for a healthcare or medical services platform, featuring a responsive design and smooth animations.

## Features

- **Home Page:** Engaging landing page with various informative sections.
- **About Us:** Details about the organization, team, and mission.
- **Services:** Comprehensive list of medical and healthcare services offered.
- **User Authentication:** Sign-in and Sign-up pages for user access.
- **Animations:** Smooth scrolling and entry animations using `framer-motion` and `react-intersection-observer`.
- **Responsive Design:** Optimized for various screen sizes.

## Technologies Used

- **Framework:** React 18
- **Routing:** React Router v6 (`react-router-dom`)
- **Styling & UI:** Vanilla CSS, `swiper` for sliders/carousels
- **Icons:** FontAwesome, Ant Design Icons, React Icons, Styled Icons
- **Animations:** Framer Motion

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Node.js and npm installed on your machine.

### Installation

1. Clone the repository (if applicable) or navigate to the project directory:

   ```bash
   cd MedCare
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\

### `npm run deploy`

Builds the application and deploys it to GitHub Pages using the `gh-pages` package.
